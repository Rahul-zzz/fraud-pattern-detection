# Fraud Pattern Detection (Rule-Based)

## Project Overview
This project detects suspicious transactions using rule-based data analytics.
No machine learning or dashboards are used.

## Objectives
- Analyze transaction data
- Define fraud detection rules
- Assign risk scores
- Classify transactions into Low, Medium, and High risk

## Rules Used
- High transaction amount (>50,000)
- New account with high amount
- Night-time transactions (12 AM â€“ 5 AM)
- International transactions with high amount

## Tools Used
- Python
- Pandas

## Output
The final output is a CSV file where each transaction is classified by risk level.
